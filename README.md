# rn-foreground-location

A React Native package for continuous location tracking while your app is active, whether in the foreground. Ideal for apps requiring real-time location updates without excessive battery drain.

## Installation

```sh
npm install rn-foreground-location
```

## Usage


```js
import { multiply } from 'rn-foreground-location'; // Import the function

// ...

const result = await multiply(3, 7);
```
